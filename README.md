# Assignment# API Automation Framework

## Overview
This project automates testing of the API:

https://jsonplaceholder.typicode.com/posts

Framework includes:
- Python
- pytest
- requests
- jsonschema
- pytest-html
- pytest-xdist

## Setup

Install dependencies

pip install -r requirements.txt

## Run Tests

pytest

## Run Tests in Parallel

pytest -n 4

## Reports

HTML report is generated automatically after execution.

report.html
